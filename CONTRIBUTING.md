# Contributing to CarbonOpus

Thank you for your interest in CarbonOpus! 

Please note that the CarbonOpus platform and this documentation repository are **proprietary and closed-source**. Our core platform code is not open to public contributions. (Our on-chain smart contracts are a separate exception).

However, we deeply value our community and beta testers! Here is how you can get involved:

## 1. Reporting Issues or Typos
If you find a bug in the documentation, a typo, or have suggestions for new guides:
*   Please open an **Issue** in this repository.
*   Provide as much detail as possible so our team can address it quickly.

## 2. Platform Feedback
For general feedback about the CarbonOpus platform (features, artist experience, fan mechanics):
*   Please reach out via our official community channels (e.g., Discord) or directly to our support team.
*   If you are part of our beta testing group, please use the designated feedback forms.

## 3. Pull Requests
Because this is a proprietary repository, **we generally do not accept unsolicited Pull Requests**. 
If you would like to suggest a significant rewrite or addition to the documentation, please open an Issue first to discuss it with the core team before investing your time in a PR.

Thank you for helping us revolutionize the music industry!
